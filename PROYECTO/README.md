# SISTEMA DE CONTROLES DE GRASS SINTETICO 
El usuario tiene gras al cual administra de manera manual, el usuario necesita que se automatize el proceso de la reserva y pago del alquiler, para lo cual solicita automatizar los siguientes casos de uso.

- el usuario podra ver la lista de horarios disponibles.
- el usuario podra reservar en unos de los horarios disponibles
- el usuario podra pagar por el alquiler de la reserva realizada
- el usuario podra vereficar su alquiler el cual le mostrar los detalles como la fceha, hora y costo del alquiler para realizar
